Webrecorder Player—Help

Webrecorder Player is a tool to browse archived versions of web pages offline.

Web archives are typically created as WARCs, a standardized format containing all kinds of web resources: HTML, Javascript, images, videos. Webrecorder player can re-assemble the pages from these resources.

Use the free service https://webrecorder.io to create WARCs, or open web archives created with other tools.*

Archive web resources you care about before they vanish!

---

Webrecorder Player supports the following web archive formats:

  • WARC format (.warc or warc.gz files) -- Webrecorder, Internet Archive, and CommonCrawl all use this web archive format
  • ARC format (.arc or .arc.gz files) -- A legacy web archiving format
  • HAR format -- HTTP Archive format
